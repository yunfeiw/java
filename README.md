## vite

下一代前端开发与构建工具

### 极速的服务启动
使用原生 ESM 文件，无需打包!

### 轻量快速的热重载
无论应用程序大小如何，都始终极快的模块热重载（HMR）

### 丰富的功能
对 TypeScript、JSX、CSS 等支持开箱即用。

### 优化的构建
可选 “多页应用” 或 “库” 模式的预配置 Rollup 构建
### 通用的插件
在开发和构建之间共享 Rollup-superset 插件接口。

### 完全类型化的API
灵活的 API 和完整 TypeScript 类型。


## 安装
使用 NPM:
```js
$ npm init @vitejs/app
```
使用 Yarn:
```js
$ yarn create @vitejs/app
```
